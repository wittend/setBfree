setBfree
========

A DSP Tonewheel Organ emulator.

setBfree is a MIDI-controlled, software synthesizer designed to imitate the
sound and properties of the electromechanical organs and sound modification
devices that brought world-wide fame to the names and products of Laurens
Hammond and Don Leslie.

see also:
 http://setbfree.org
 https://github.com/pantherb/setBfree


Application Installation
------------------------

Drag/drop setBfree.app to the /Application folder.
The standalone application depends on JACK - http://jackaudio.org for
audio I/O.


Plugin Installation
-------------------

Simply run the included LV2Installer.app, it will ask to do either
of the following for you:

Copy the the b_synth.lv2 and b_whirl.lv2 folders to
  $HOME/Library/Audio/Plug-Ins/LV2/
or install system-wide for all users to
  /Library/Audio/Plug-Ins/LV2/
